Rủi ro bảo mật · OpenClaw

Tăng trưởng quá nhanh để kiểm soát an toàn

ClawHub Supply Chain Risk

341 malicious skills được upload lên ClawHub (Jan–Feb 2026). Bất kỳ GitHub account nào >1 tuần tuổi có thể publish skill — không có code review.

35.4% Instances Bị Lỗ Hổng

SecurityScorecard: 40,214 instances exposed trên internet. 4,500+ cài đặt sai lộ API keys và passwords. CVEs: command injection, SSRF, path traversal, prompt injection.

Phản ứng của OpenClaw

Đã công bố Five-Point Security Plan nhưng từ chối cam kết "risk-free AI agent". Microsoft Security Blog đã đưa ra hướng dẫn riêng về isolation và runtime risk.

Hermes vs OpenClaw Security

Hermes có 0 CVE — nhưng đây vì Hermes còn mới và ít được test, không phải vì bảo mật tốt hơn căn bản. Khi Hermes scale lên, rủi ro sẽ xuất hiện.

Tiềm năng & Tương lai

OpenClaw: Scale — Hermes: Intelligence

OpenClaw · Tiềm năng
  • ↗ Foundation governance mở đường cho enterprise adoption quy mô lớn — B2B integrations, managed hosting, consulting ecosystem.
  • ↗ OpenAI backing: tiềm năng tích hợp sâu với các model mới nhất của OpenAI, tools, và APIs.
  • ↘ Rủi ro: Security crisis có thể làm chậm enterprise adoption. ClawHub cần curated marketplace.
Hermes · Tiềm năng
  • ↗ GEPA là lợi thế không thể copy ngay — nếu self-improvement loop chứng minh compounding gains trên public benchmarks, game-changer thật sự.
  • ↗ Model-agnostic + open weights Hermes LLMs = fully sovereign deployment cho enterprise có yêu cầu data privacy.
  • ↘ Rủi ro: Learning loop còn bias "self-congratulation" — agent đánh giá performance của mình quá tốt. Chưa có evidence compounding gains trên scale lớn.

💡 Chiến lược tối ưu 2026: Chạy OpenClaw làm orchestrator (messaging + routing) + Hermes làm execution specialist (research, analysis, phức tạp). Cả hai MIT license, cả hai self-hostable.

Kết luận · Takeaways

OpenClaw thống trị hôm nay.
Hermes đặt cược vào ngày mai.

🏆

OpenClaw wins

Ecosystem, community, integrations. Dùng ngay nếu cần orchestration platform cho messaging + multi-platform workflows.

🧠

Hermes wins

Self-improvement và persistent memory. Dùng nếu cần automation ngày càng thông minh hơn theo thời gian — không cần setup lại từ đầu.

⚡

Dùng cả hai

OpenClaw = control plane. Hermes = AI brain. Cả hai MIT, cả hai open-source, cả hai self-hostable. Stack này beat mọi thứ SaaS.
